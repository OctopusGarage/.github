# OctopusGarage

> 🧠 Open source tools for self-discovery — cognitive assessment and personality science, made accessible.

Psychometrics made beautiful. We're building free, scientifically-backed self-assessment tools that anyone can use — no sign-up, no paywall, no fluff.

---

## Projects

### [IQ Lab](https://github.com/OctopusGarage/cognitive-lab)

> Professional IQ test based on Raven's Progressive Matrices

A 40-question cognitive assessment using IRT (Item Response Theory) for precise IQ estimation. Bilingual (Chinese / English), PWA, offline-capable.

🔗 [Try it →](https://octopusgarage.github.io/cognitive-lab/)

---

### [MBTI Lab](https://github.com/OctopusGarage/mbti-lab)

> Professional MBTI personality test

Discover your 16-type personality profile with rich results — type descriptions, strengths, career matches, and famous people. Bilingual, shareable, no sign-up required.

🔗 [Try it →](https://octopusgarage.github.io/mbti-lab/)

---

## Principles

- **Scientifically grounded** — methodology drawn from validated psychometrics (Raven's RPM, MBTI, IRT)
- **Privacy first** — all data stays in your browser, zero tracking
- **Beautiful by default** — design that respects the subject matter
- **Open source** — transparent methodology, freely available

---

## Tech

Both projects are built with **Vue 3 + Vite**, deployed to GitHub Pages, and support offline use as PWA installable apps.
